Windows XP Embedded, atau XPe, adalah versi dikecilkan Microsoft Windows XP Professional dan pengganti kepada Windows NT 4.0 Embedded. Ia berasaskan inti yang sama dengan Windows XP Professional tetapi dipasarkan kepada pembangun OEM, ISV dan IHV yang memerlukan fungsi penuh 32 bit Windows tanpa memerlukan perkakasan Windows XP Professional. Ia boleh menjalankan aplikasi Windows serta fail pemacu peranti pada peranti dengan sekurang-kurangnya 32 MB ingatan kilat CompactFlash, 32 MB RAM dan mikropemproses P-200. XPe dilancarkan pada 28 November 2001. Setakat Februari 2007, keluaran terbaru XPe adalah Windows XP Embedded SP2 Feature Pack 2007. XPe tiada kaitan dengan Windows CE. Masing-masing mensasarkan peranti yang berbeza dan masing-masing mempunyai kelebihan serta kelemahan tersendiri yang menarik minat pengeluar OEM tertentu bagi peralatan tertentu.
